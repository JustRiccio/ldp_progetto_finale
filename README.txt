Nonostante questa suddivisione, ogni parte del progetto è stata discussa insieme e tutto

NOTE AGGIUNTIVE:
-SpegnimentoDispositivo: non è stata implementata la "priorità" dei dispositivi(es. Il frigo non può essere spento per dare spazio ad altri dispositivi), 
 perchè avrebbe richiesto un'implementazione particolarmente artificiosa nel caso in cui si avesse voluto risolvere il
 problema in modo elegante (es.: avere una lista di dispositivi con una diversa priorità salvata su file).
 Se è necessario spegnere dispositivi, l'unica priorità è data dal tempo di accensione.
